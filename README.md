# Syntax Club - Official Club Management Platform

Welcome to the official repository for the Syntax Club application. This is a full-stack web platform designed to manage all aspects of the club's activities, from member onboarding to event management and community engagement.

## âœ¨ Features

- **Member Management**: A complete system for member registration, profiles, and role management (Admin, Member, Leader).
- **Event & Ticketing System**: Create and manage club events, handle registrations, and generate unique QR-code based tickets.
- **Annual Tech Fest Hub ("Arvantis")**: A dedicated module to manage our flagship annual tech fest, including events, partners, sponsors, and media galleries.
- **Application Portal**: A streamlined process for prospective members to apply to join the club.
- **Community Engagement**: Features for social posts and a public-facing contact system.
- **Secure Admin Dashboard**: A protected area for admins to manage all club data, view analytics, and oversee operations.
